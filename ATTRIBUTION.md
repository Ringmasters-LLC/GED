# Attribution

This project is derived from and compatible with several open reference datasets.

## Data Sources

- **ISO 3166**: Country codes and names.
- **Google libphonenumber**: Phone metadata and calling codes.
- **CLDR (Common Locale Data Repository)**: Locale, territory, language, and currency data.
- **GeoNames**: Postal data and administrative subdivisions.

Please see `DATA_SOURCES.md` for detailed licensing and attribution requirements per source. Every row in the distributed data includes a `sources` array tracing it back to its origin.
